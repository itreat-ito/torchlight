ローカル・テスト・本番環境を行き来しながら行う開発・制作作業をサポートするChrome拡張機能です。

## 機能

- **各環境のバナー表示**  
ドメインに基づいて現在の環境(ローカル・テスト・本番)を自動判定し、画面上部にバナーで表示する
- **環境の切り替え**  
現在開いているページに対応する他環境のURLへ切り替える
- **ページタイトルのプレフィックス**  
各環境のページタイトル先頭に任意のテキストを付与する
- **クリップボードへのコピー**  
ページタイトルとURLを指定のフォーマットでクリップボードにコピーする

## 初回インストール手順

### 1. zipファイルを解凍

1. [配布ページ](https://github.com/itreat-ito/torchlight/releases/latest)のAssets欄にあるzipファイルをダウンロードし、任意の場所で解凍してください。
2. 解凍してできたフォルダの名称からバージョン番号を削除してください。  
例) torchlight-1.x.x → torchlight

### 2. Chromeの拡張機能管理画面を開く

1. Chromeブラウザを開いてください。
2. アドレスバーに `chrome://extensions/` と入力してEnterキーを押してください。拡張機能管理画面が開きます。
   - またはアドレスバー右端の三点リーダメニューから「拡張機能」→「拡張機能を管理」と進んでください。

### 3. デベロッパーモードを有効にする

拡張機能管理画面の右上にある「デベロッパーモード」のスイッチをオンにしてください。

### 4. 拡張機能を読み込む

1. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックしてください。
2. 「1. 拡張機能ファイルを解凍」で作成したフォルダを選択してください。

### 5. インストール完了

拡張機能がインストールされ有効になります。


## 最新バージョンへの更新手順

新しいバージョンが配布された場合の更新手順です。

1. [配布ページ](https://github.com/itreat-ito/torchlight/releases/latest)のAssets欄からzipファイルをダウンロード・解凍して、古いバージョンのフォルダと**同じ名称**で上書きしてください。
2. Chromeの拡張機能管理画面（`chrome://extensions/`）を開いてください。
3. torchlightの「再読み込み」ボタンをクリックしてください。
4. バージョン番号が[配布ページ](https://github.com/itreat-ito/torchlight/releases/latest)に記載されている番号と同じになっていれば更新成功です。 

**注意**: 拡張機能自体を削除すると設定内容が失われてしまいます。必ず上記の手順で更新してください。
