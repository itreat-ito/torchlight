# Chrome拡張機能のインストール方法

この拡張機能をChromeにインストールする手順です。

## インストール手順

### 1. 拡張機能ファイルを解凍

配布されたzipファイルを解凍してください。

### 2. Chromeの拡張機能管理画面を開く

1. Chromeブラウザを開きます
2. アドレスバーに `chrome://extensions/` と入力してEnterキーを押す
   - または、メニュー（三点リーダー）→「その他のツール」→「拡張機能」を選択

### 3. デベロッパーモードを有効にする

拡張機能管理画面の右上にある「デベロッパーモード」のトグルスイッチをONにします。

### 4. 拡張機能を読み込む

1. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
2. 解凍したフォルダ（`dist`フォルダ）を選択
3. 「フォルダーの選択」をクリック

### 5. インストール完了

拡張機能がインストールされ、有効化されます。拡張機能のアイコンがChromeのツールバーに表示されます。

## 拡張機能の更新方法

新しいバージョンが配布された場合の更新手順です。

### 方法1: 再読み込みボタンを使用（推奨）

1. Chromeの拡張機能管理画面（`chrome://extensions/`）を開く
2. 更新する拡張機能の「再読み込み」ボタン（🔄アイコン）をクリック
3. 新しいバージョンのzipファイルを解凍して、**同じ場所**に上書きする
4. 再度「再読み込み」ボタンをクリック

**重要**: この方法では、既存の設定（プロジェクト情報やカラー設定など）は**すべて保持**されます。

### 方法2: フォルダを再選択

1. 新しいバージョンのzipファイルを解凍して、**同じ場所**に上書きする
2. Chromeの拡張機能管理画面（`chrome://extensions/`）を開く
3. 拡張機能の「詳細」をクリック
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. **同じフォルダ**を再度選択する

**注意**: 拡張機能を**削除せずに**同じフォルダを再読み込みすることで、設定が保持されます。

## 設定の保持について

- ✅ **設定は保持されます**: 拡張機能を削除せずに更新する限り、以下の設定は保持されます
  - プロジェクト情報（ローカル/ステージング/本番のURL設定）
  - バナーのテキストとカラー設定
- ⚠️ **設定が失われる場合**: 拡張機能を削除してから再インストールすると、設定は失われます

## トラブルシューティング

- **エラーが表示される場合**: 一度拡張機能を削除してから、再度読み込んでください
- **アイコンが表示されない場合**: 拡張機能管理画面で拡張機能が有効になっているか確認してください
- **動作しない場合**: Chromeを再起動してみてください
